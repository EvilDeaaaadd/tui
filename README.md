# tui

Console based widget toolkit for building terminal user interfaces

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     tui:
       github: SamualLB/tui
   ```

2. Run `shards install`

## Usage

```crystal
require "tui"

# require one of the following backends
require "tui/backend/ncurses"
# add more backends...
```
